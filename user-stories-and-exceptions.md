User Stories

1. As a user, I want to create an account, so that I can post and manage my reviews.
2. As a user, I want to log in to my account, so that I can access my personal review history.
3. As a user, I want to search for a restaurant by name, so that I can quickly find reviews for a specific place.
4. As a user, I want to browse a list of restaurants in the database, so that I can discover places to eat.
5. As a user, I want to add a new restaurant to the database, so that other users can review it.
6. As a user, I want to view details about a restaurant, so that I can learn more before deciding to visit.
7. As a user, I want to read reviews written by other users, so that I can see their opinions and experiences.
8. As a user, I want to give a restaurant a rating, so that I can share how good or bad my experience was.
9. As a user, I want to write a review for a restaurant I visited, so that I can describe my experience.
10. As a user, I want to edit a review I posted, so that I can correct or update my opinions later.
11. As a user, I want to delete one of my reviews, so that I can remove feedback I no longer want published.
12. As a user, I want to see all reviews I have written, so that I can keep track of my activity.
